Development on Veilkeeper is starting with the core tactical layer. The goal is to build the game from the combat foundation outward, making sure each new system has something solid to sit on.

Right now, the focus is on establishing the grid, input handling, and the basic structure of turn-based interactions. This stage is less about visuals and more about making sure the game feels responsive and predictable.

The early prototype is centered around clarity. Movement, positioning, and board state should always be easy to understand at a glance. Even small improvements here make a big difference later when abilities, enemies, and systems begin stacking on top.

## Current progress in this stage:

- Initial tactical grid setup
- Basic board interaction
- Early input handling
- Foundation for turn-based flow

The next step is improving cursor control and interaction across different input methods. Once the board can be navigated cleanly, the focus will shift toward selecting units and moving them across the grid.
