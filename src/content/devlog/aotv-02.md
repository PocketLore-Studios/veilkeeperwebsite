Veilkeeper’s visual direction is starting to take shape through its UI elements. This set focuses on core icons—equipment, resources, and consumables—used throughout the game.

Earlier work was primarily grayscale to focus on value and readability. This pass introduces color into the system, specifically along the amber and purple spectrum that defines Veilkeeper’s world. Amber represents stability and the home shard, while purple reflects the instability of the rift.

The goal with this set wasn’t just to produce individual assets, but to begin building a cohesive visual language. Each icon is designed to read clearly at small sizes while still reinforcing material differences such as metal, glass, and crystal. Consistency across these elements is key, as they will appear repeatedly in the UI.

There’s still work to do—particularly in pushing contrast and improving material definition—but this is the first group of assets that feels like it belongs to the same system rather than isolated pieces.

## Current Progress

- Introduced color into UI icon pipeline
- Established amber/purple palette direction
- Created initial set of icons (equipment, potions, crystal)
- Focused on readability at small scale
- Beginning to define consistent visual language
